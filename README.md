# Real-Time-Facial-Emotion-Recognition-using-DeepFace-OpenCV
This project performs real-time facial emotion detection using your webcam. It uses the DeepFace library—an advanced facial analysis framework—to identify the dominant emotion in each video frame and display it directly on screen.

Features

Real-time emotion recognition using a webcam

Detects multiple emotions such as:

happy, sad, angry, surprised, neutral, fearful, disgust, contempt

Displays the dominant emotion directly on the video feed

Runs fully on CPU (GPU optional)

Built using DeepFace’s pre-trained models

Tech Stack

Python

OpenCV

DeepFace (TensorFlow backend)
